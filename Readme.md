# Simple React App to consume Currency API

## Clone Repository
```git clone https://github.com/woodenfloor-3/CurrencyAPI.git```

## Install Dependencies
```npm install```

## Install Tailwind CSS and setup
```npm install -D tailwindcss@3.4.19 postcss@8.5.6 autoprefixer@10.4.24```

```npx tailwindcss init -p```

## ENV
``` VITE_API_BASE_URL= "http://<ipaddr:port>"```

## Run
```npm run dev```
